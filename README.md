Features

💬 Text Mode — Type your doubts, get sharp punchy replies. Works on any device, any browser.

🎙️ Voice Mode — Speak your excuse out loud. DoubtZero X listens, thinks, and talks back. Hands-free, fully conversational.

Voice Capabilities: Web Speech API for native voice input (Speech Recognition) and voice output (Speech Synthesis).

How to Deploy (Vercel)
DoubtZero X is built for a seamless, zero-configuration deployment on Vercel.

Prerequisites:

A free GitHub account.

A free Vercel account.

A Google Gemini API Key.

Steps:

Upload the 4 project files (index.html, vercel.json, package.json, and the api folder) to a new GitHub repository.

Log in to Vercel and click Add New -> Project.

Import your GitHub repository.

Open the Environment Variables section before clicking deploy. Add a new variable:

Key: GEMINI_API_KEY

Value: (Paste your Google Gemini API key here)

Click Deploy.

Vercel will automatically route your frontend and secure your /api/chat.js backend. Your DoubtZero X session is now live.
